# Chinese-Find 智慧中文搜尋

**自動識別繁、簡、異體字，讓您的頁面搜尋（`Ctrl+F`）再無障礙。**

您是否曾在使用 `Ctrl+F` 搜尋網頁時，因為繁簡或異體字的差異而找不到想要的內容？這個瀏覽器擴充功能就是為解決了這個問題而生。它讓您輸入關鍵字，即可同時搜尋其對應的繁體、簡體及異體字。

---

## ✨ 主要特色

- **智慧識別**：輸入關鍵字，即可同時搜尋其對應的繁體、簡體及異體字。
- **自訂樣式**：您可以自由調整搜尋結果的背景顏色、透明度，以及當前匹配項的外框樣式。
- **非破壞性**：不修改原始網頁結構達到高亮效果，確保網頁功能穩定不受影響。
- **無縫整合**：透過您熟悉的 `Ctrl+F` 或自定義快捷鍵即可啟用。
- **整頁轉換**：內建一鍵將整個網頁轉換為繁體或簡體中文的便利工具。
- **開源**：所有程式碼皆公開於 GitHub，可供審查。

## 📖 使用教學

1. 按下 `Ctrl+F` 開啟搜尋面板。
2. 輸入您想尋找的中文關鍵字。
3. 按下 `Enter` 或面板上的箭頭按鈕，在匹配結果之間輕鬆跳轉。
